## 문제 
아래 예제와 같이 새싹을 출력하시오.
## 입력
입력은 없다.
## 출력
새싹을 출력한다.
## 예제 입력 
```

```

## 예제 출력  
```
         ,r'"7
r`-_   ,'  ,/
 \. ". L_r'
   `~\/
      |
      |
```
## 코드
```
#include <iostream>
#include <string>

int main(){
    std::string s;

    s = "         ,r'\"7\n";
    s += "r`-_   ,'  ,/\n";
    s += " \\. \". L_r'\n";
    s += "   `~\\/\n";
    s += "      |\n";
    s += "      |";
    
    std::cout << s;
}
```
## 풀이
앞서 풀었던 입출력 문제와 동일합니다.

```"```와 ```\```을 출력할 때만 주의해주면 됩니다.

cout로 바로 출력해도 되지만 string 문자열을 활용하여 문자열에 저장하고 출력하는 방식을 사용하였습니다.
