## 문제 
아래 예제와 같이 고양이를 출력하시오.

## 입력
없음.


## 출력
고양이를 출력한다.

## 예제 입력 
```

```

## 예제 출력  
```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

## 코드
```
#include <iostream>

int main() {
    std::cout << "\\    /\\\n";
    std::cout << " )  ( ')\n";
    std::cout << "(  /  )\n";
    std::cout << " \\(__)|\n";
    return 0;
}
```

## 풀이
```\``` 해당 기호를 출력하려면 앞에 "\" 기호를 붙여서 출력할 수 있음.
