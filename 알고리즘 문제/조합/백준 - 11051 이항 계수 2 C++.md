## 문제 
자연수 
\(N\)과 정수 
\(K\)가 주어졌을 때 이항 계수 
nCk를 10,007로 나눈 나머지를 구하는 프로그램을 작성하시오.
## 입력
첫째 줄에 
\(N\)과 
\(K\)가 주어진다. (1 ≤ 
\(N\) ≤ 1,000, 0 ≤ 
\(K\) ≤ 
\(N\))
## 출력
nCk를를 10,007로 나눈 나머지를 출력한다.


## 예제 입력 
```
5 2
```

## 예제 출력  
```
10
```
## 코드 풀이
```python
#include <iostream>
using namespace std;

static int n, k;
static int D[1001][1001];

int main(){
    cin >> n >> k;
    
    for(int i = 0; i <= n; i++){
        D[i][1] = i;  // i개 중 1개를 뽑는 경우의 수 i개
        D[i][0] = 1;  // i개 중 1개도 뽑지 않는 경우의 수 1개
        D[i][i] = 1;  // i개 중 i개를 뽑는 경우의 수 1개
    }
    
    for(int i = 2; i <= n; i++){
        for(int j = 1; j < i; j++){
            D[i][j] = D[i - 1][j] + D[i - 1][j - 1];  // 점화식
            D[i][j] %= 10007;
        }
    }
    
    cout << D[n][k] << '\n';
}
```
